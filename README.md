# Laboratory 2 - Project 1A

Dynamic vector index implementation and local evaluation scripts.

## Requirements

- Python 3
- NumPy

Install the dependency:

```bash
python3 -m pip install -r requirements.txt
```

## Run a public scenario

The public scenario files are stored locally under `data/public/` and are not
committed because some of them exceed GitHub's file-size limit.

```bash
python3 scripts/eval_scenario_1.py
python3 scripts/eval_scenario_2.py
python3 scripts/eval_scenario_3.py
```
